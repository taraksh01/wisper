Powerful Features

Everything you need for seamless voice dictation on Linux

Global Hotkey

Press Shift+Space from anywhere to instantly start or stop recording.

AI Transcription

Powered by OpenAI Whisper via Groq or OpenAI for accurate results.

Clipboard Integration

Transcribed text is automatically copied to clipboard for easy pasting anywhere.

Minimal UI

Slim, transparent recording bar with real-time audio waveform visualization.

System Tray

Quick access from system tray. Toggle recording and access settings.

Privacy First

Records locally before sending to API. No data stored on external servers.

Get Started

Set up Wisper in under 5 minutes

1

Download Wisper

Get the AppImage (universal) or .deb package for Debian/Ubuntu from the download section below.

2

Install & Run

AppImage: Make executable and double-click
.deb: Install via Software Center or sudo dpkg -i wisper_*.deb

3

Get Your API Key

Choose a provider and get your free API key:

Groq Console Free OpenAI Platform
4

Configure & Start

Right-click system tray icon → Select Settings → Choose provider → Paste API key → Click Save Settings → Shift+Space to dictate!
